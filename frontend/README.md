# Enterprise RAG Frontend

Next.js frontend for the Enterprise RAG Platform — a multi-tenant document chat and retrieval system.

## Setup

1. Copy environment variables:
   ```bash
   cp .env.example .env.local
   ```

2. Update `.env.local` with your Supabase and backend URLs.

3. Install and run:
   ```bash
   npm install
   npm run dev
   ```

## Tech Stack

- Next.js App Router
- TypeScript
- Tailwind CSS + shadcn/ui
- Supabase Auth
- Lucide React icons

See the root [README.md](../README.md) for full project setup instructions.
